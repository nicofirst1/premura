# Architecture Decision Records

ADRs for Premura live in this directory.

- File names use sequential numbering: `0001-slug.md`, `0002-slug.md`, and so on.
- Create an ADR only for a decision that is hard to reverse, surprising without context, and the result of a real trade-off.
- Keep ADRs short. A single paragraph is enough when it captures the context, the decision, and why it was made.

Template:

```md
# Short title of the decision

One to three sentences covering the context, the decision, and why it won.
```
